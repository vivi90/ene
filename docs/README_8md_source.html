4 About

14 Naming
15 ------
16 We have named it after the character 'Ene' from the anime 'Mekakucity Actors'.
17 
18 Usage
19 -----
20 Just compile it or use the attached binary file `bin/ene.jar`.
21 Currently it supports only WAV files.
22 The library database is saved in a local SQLite file, called `library.db`.
23 The playlist file is called `playlist.m3u`.
24 
25 Features
26 --------
27 * Create a simple music library
28 * Play tracks
29 * Sort tracks by artist, title or genre (ascending or descending)
30 * Filter tracks by artist, title or genre
31 * Create a playlist
32 * Currently available translations: English & German

48 Troubleshooting
49 ---------------
50 If you run this application on a Linux operating system, the sound doesn't work,
51 and you get an `LineUnavailableException` in debug mode, then please force Java to use the Pulse Audio system by issuing: `padsp java -jar bin/ene.jar`
52 You need to start this application with the `-d` argument to enable debug infos.
