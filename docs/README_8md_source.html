4 About
5 -----
6 This project is a simple extensible music library as a homework in the context of my computer science studies.
7 For more details about the source code please check out the documentation: https://vivi90.github.io/ene
8 
9 Naming
10 ------
11 We have named it after the character 'Ene' from the anime 'Mekakucity Actors'.
12 
13 Usage
14 -----
15 Just create a `Music` folder at the project root directory and compile it.
16 Currently it supports only WAV files.

31 Troubleshooting
32 ---------------
33 If you run this application on a Linux operating system, the sound doesn't work,
34 and you get an `LineUnavailableException` in debug mode, then please force Java to use the Pulse Audio system by issuing: `padsp java -jar bin/ene.jar`
35 You need to start this application with the `-d` argument to enable debug infos.
